README
R code to run all simulations and analyse presented in Puttick et al. 

FILE

install.RSOLE.R
	For all R code it is necessary to install the RSOLE package from GitHub
	
FOLDERS

run.simulations
	R code to run all simulations to generate trees and trait data without mass
	extinctions, with mass extinctions, and with trait-biased background speciation and extinction 
	
precooked.output
	R code to run all simulations to generate trees and trait data without mass extinctions,
	with mass extinctions, and with trait-biased background speciation and extinction 
	
disparity.analysis
	R code to carry-out main analysis on simulated data
